# Copilot Task Poller

A Copilot agent to autonomous process coding tasks.

## Usage

1. Copy `.github` folder into your project.
2. Open Copilot Chat in Visual Studio Code, choose "claude-opus-4.6" model, type "#codebase start task-poller", enter.
3. Add tasks/questions/comments into `TASKS.md` file and watch the agent works.
4. If the agent finishes, he's gonna respond both in Chat UI and `TASKS.md` file.
5. Back to step 3 unless done or terminated due to networking issues.

## Q&A 

1. Can I use in Copilot CLI? - Yes, same mechanism.
2. Why "claude-opus-4.6"? - It's smart enough to strictly follow the instructions.
